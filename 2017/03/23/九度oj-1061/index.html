九度oj-1061

发表于 2017-03-23   |   分类于 九度oj   |  

题目描述：

有N个学生的数据，将学生数据按成绩高低排序，如果成绩相同则按姓名字符的字母序排序，如果姓名的字母序也相同则按照学生的年龄排序，并输出N个学生排序后的信息。

输入：

测试数据有多组，每组输入第一行有一个整数N（N<=1000），接下来的N行包括N个学生的数据。
每个学生的数据包括姓名（长度不超过100的字符串）、年龄（整形数）、成绩（小于等于100的正数）。

输出：

将学生信息按成绩进行排序，成绩相同的则按姓名的字母序进行排序。
然后输出学生信息，按照如下格式：
姓名 年龄 成绩

样例输入：

3
abc 20 99
bcd 19 97
bed 20 97

样例输出：

bcd 19 97
bed 20 97
abc 20 99

代码

#include<stdio.h>  
#include<algorithm>  
#include<string.h>//忘记写这句话了  
using namespace std;  
 struct student  //为什么开始加着typedef不可以呢?  
{  
    char name[101];//不知道台这里为什么是开始己写的是100  
    int age;  
    int score;  
}buf[1000];  
 bool cmp(student a , student b)//注意观察?类题使用的模板  sort函数的巧妙使用，
    //bool cmp比括较函数的使用  
 {  
     if(a.score != b.score)  
     {  
         return a.score < b.score;  
     }  
     int tmp = strcmp(a.name, b.name);//!!!!!!!important  
     if(tmp !=0) return tmp<0; //重点理解     
     else return a.age<b.age;  
 }  
int main()  
{  
    int n;  
    //int num[1000];  
    //用不着重复声明了?  
    //student student[];  
    while( scanf("%d", &n) != EOF )  
    {  
        for(int i=0; i<n; i++)  
        {  
            scanf("%s %d %d", &buf[i].name,&buf[i].age, &buf[i].score);  
        }  

        sort(buf , buf+n, cmp);  

        for(int i=0; i<n; i++)  
        {  
            printf("%s %d %d\n", buf[i].name, buf[i].age, buf[i].score);  
        }  

    }  
    return 0;  
}

作者提醒

也可以直接定义该结构体的 小于运算符 来说明排序规则

Bool operator < (const E &b) const {}

这样定义 < 运算符 后，调用sort函数时，可以直接sort(buf , buf+n); 不用第三个参数了。

原因是sort函数只利用< 运算符来定序，小的在前面。 
所以在调用sort函数时，不必特别指明排序规则（就是说不适用第三个参数了）
